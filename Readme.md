# Sales Performance Analytics Dashboard

## Overview
An end-to-end Sales Analytics Dashboard built using MySQL and Power BI.

## Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales by Region
- Sales Trend Analysis
- Top 10 Products
- Sales by Category
- Sales by Segment
- Interactive Slicers

## Tech Stack
