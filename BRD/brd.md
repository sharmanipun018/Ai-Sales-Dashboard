# Business Requirements Document (BRD)

## Project Name
Sales Performance Analytics Dashboard

## Objective
Build an interactive dashboard to analyze sales performance and generate business insights using MySQL and Power BI.

## Business Requirements
1. Track Total Sales.
2. Track Total Profit.
3. Track Total Orders.
4. Analyze sales by Region.
5. Identify Top 10 Products.
6. Analyze sales by Category.
7. Analyze sales by Segment.
8. Monitor Sales Trends over time.
9. Provide interactive filtering using slicers.

## KPIs
- Total Sales
- Total Profit
- Total Orders

## Tools Used
- MySQL
- Power BI
- VS Code

## Expected Outcome
A dashboard that helps users understand sales performance and make data-driven decisions.
